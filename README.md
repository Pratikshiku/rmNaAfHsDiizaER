## University Town Water and Electricity Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1648cab4a5b646efb1d4778e0c5977ae/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Center
5. Equipment Collection
6. Equipment Consumption
7. Equipment Application
8. Equipment Distribution
9. Collection Report
10. User Feedback
11. Repair Information


**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Equipment Collection Management
5. Equipment Consumption Management
6. Equipment Application Management
7. Equipment Distribution Management
8. Status Report Management
9. Collection Report Management
10. Consumption Report Management
11. Team Report Management
12. Personal Report Management
13. User Feedback Management
14. Maintenance Management
15. Equipment Inspection Management
16. Equipment Repair Management
17. Repair Information Management
18. Regular Repair Management
19. Repair Plan Management
